## Useful Beginner Stack
- [LangGraph](https://docs.langchain.com/oss/python/langgraph/overview) – Builds agent logic and memory
- [OpenRouter](https://openrouter.ai/) – One API key for all models
- [Tavily](https://tavily.com/) – Web search built for agents
- [Firecrawl](https://www.firecrawl.dev/) – Turns websites into readable text
- [Chroma](https://docs.trychroma.com/) – Local database for agent memory
- [Vercel](https://vercel.com/) – Easy hosting for your app
