# Intro to React Development: Your First Web Application

**Company:** EOG

**Time:** Saturday, 5:30pm - 6:15pm

**Location:** ECSW 1.355

---

## Workshop Description

Learn how modern web apps come to life with React! In this hands-on session, you’ll build your first React application while exploring core concepts like components, state, and props. By the end, you’ll have a working app — and the skills to create interactive dashboards and tools for your hackathon project.
